# Viscometer

**Viscometer** — приложение для моделирования работы вискозиметра в рамках лабороторной работы РТУ МИРЭА.

## Требования

- **Qt 6**
- **CMake**
- **clang/gcc C++20**

## Подготовка к сборке

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Avinar16/Viscometer.git
   cd Viscometer
   mkdir build && cd build
   cmake ..
   ```
# Cборка
```bash
make
```
